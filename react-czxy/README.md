### 目录结构描述
```
├──app              主目录
│   ├──component    react组件目录
│   └──index        首页目录
│ 
├──public           公共目录 主要放 js css img
│   ├──js           公共js文件
│   ├──css          图片目录
│   └──img          图片目录
│
├──devBuild         webpack 开发环境下 打包输出目录
│
├──config           打包配置目录
│
├──webpack          webpack配置目录
│
├──entryBuild       webpack 打包入口文件目录

```
